# chess-python-game
Jeu d'échecs en Python avec interface graphique et pendules

## Description
Mon jeu d'échecs est un projet développé en Python. Il permet aux joueurs d'entrer leurs noms et d'utiliser des pendules (clocks) pour gérer le temps de jeu.
